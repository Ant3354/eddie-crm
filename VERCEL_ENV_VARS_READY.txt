INSTRUCTIONS
========================================

1. Go to vercel.com
2. Create account / Login
3. Click "New Project"
4. Import your GitHub repository
5. Go to "Environment Variables"
6. Add each variable above
7. Make sure to select BOTH "Production" and "Preview"
8. Click "Deploy"
9. Wait for build (2-3 minutes)
10. Copy your deployment URL
11. Update NEXT_PUBLIC_APP_URL to your actual URL
12. Redeploy
